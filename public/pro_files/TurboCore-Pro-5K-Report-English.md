## 🧠 How TurboCore Pro Works

TurboCore Pro runs an **advanced multi-asset ML allocation system** with daily regime-aware portfolio rebalancing.

### The 4-Asset Intelligent Mix

Unlike traditional strategies that hold one asset, TurboCore Pro dynamically blends:

| Asset | Description | Use Case |
|-------|-------------|----------|
| **QQQ** | NASDAQ-100 ETF (1x) | Core equity exposure, defensive positioning |
| **QLD** | ProShares Ultra QQQ (2x) | Moderate leverage in confirmed trends |
| **QQQ LEAPS** | Deep ITM Call Options (Delta 0.80-0.90) | Maximum leverage with controlled risk in bull regimes |
| **SGOV** | 0-3 Month T-Bills | Cash preservation during bear markets |

### Step 1 — Regime Detection (HMM + ML)

The AI classifies the market into one of 3 states using Hidden Markov Models + XGBoost:

| State | What it means | What TurboCore Pro does |
|-------|--------------|------------------------|
| 🟢 **BULL** | Strong uptrend, low volatility | Deploy QQQ LEAPS (60%+) for maximum capture |
| 🟡 **SIDEWAYS** | Choppy, uncertain | Rotate to QQQ + QLD, reduce LEAPS |
| 🔴 **BEAR** | Breakdown detected | 100% SGOV (T-bills earning 4-5%) |

### Step 2 — Signal Confidence Scoring

An XGBoost classifier scores every signal from 0-100% confidence:
- Score **≥ 75%** = Aggressive LEAPS allocation (40-60%) 🚀
- Score **60-75%** = Moderate LEAPS + QLD blend ⚖️
- Score **< 60%** = Conservative QQQ + SGOV positioning 🛡️

### Step 3 — Dynamic LEAPS Position Management

The system automatically:
- **Selects optimal delta** (0.80-0.90) based on IV regime
- **Times entry/exit** using regime transitions
- **Rolls positions** at optimal IV windows
- **Sizes allocation** using Kelly Criterion (1/4 size for safety)

### Step 4 — Daily Fractional Rebalancing

TurboCore Pro executes **micro-adjustments** every trading day:
- Sells portions of assets no longer matching regime
- Buys exact fractional shares to hit target allocation
- Total of **4,599 order executions** over 7 years = precision rotation
